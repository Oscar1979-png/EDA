# EDA
EDA on Social Media Addiction

EDA: Adicción a las Redes Sociales en Estudiantes Universitarios
Este repositorio contiene un Análisis Exploratorio de Datos (EDA) detallado sobre el dataset SMA10 (Social Media Addiction), enfocado en identificar patrones de comportamiento y niveles de dependencia digital en la población estudiantil universitaria.


Sobre el Dataset
El dataset SMA10 es un recurso público diseñado específicamente para cuantificar la adicción a las plataformas digitales mediante métricas psicométricas y hábitos de uso.
Fecha de publicación: 6 de enero de 2025 (Versión 1).
DOI: 10.17632/9mxm455dfm.1
Población: Estudiantes universitarios.

Objetivo: Evaluar la prevalencia y los factores de riesgo asociados a la adicción a redes sociales.
Objetivos del Análisis
Perfilamiento: Identificar las características demográficas predominantes en la muestra.
Patrones de Uso: Analizar el tiempo de permanencia y las plataformas más utilizadas.
Detección de Riesgo: Clasificar los niveles de adicción según la escala SMA10.
Correlaciones: Encontrar vínculos entre el rendimiento académico, el bienestar emocional y el uso de redes.

Tecnologías Utilizadas
Python 3.x
Pandas / NumPy: Manipulación y limpieza de datos.
Matplotlib / Seaborn: Visualización de datos estadística.
Jupyter Notebook: Entorno de desarrollo del análisis.
